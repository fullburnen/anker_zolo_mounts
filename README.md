# Anker Zolo Qi2 Wireless Charger Mounts

3 mounts for the Anker Zolo Qi2 A25M2.

* Desk edge mount with clamp
  * Max thickness 20mm
  * Secured using a M12 bolt
* Desk edge mount without clamp
  * For use with adhesive
* Flat surface mount
